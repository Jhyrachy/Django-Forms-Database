# Database Manager for Patient Forms

## Overview
This project is designed to provide an easy-to-use and configurable database manager for handling patient forms. It allows users to add new patients and edit their data in various settings. The system is optimized to load only necessary data at a time, ensuring that the interface remains user-friendly and not overwhelming.

## Features
- **Add New Patients**: Easily add new patient records to the database.
- **Edit Patient Data**: Modify existing patient information across different settings.
- **Optimized Data Loading**: Load only relevant data to keep the interface clean and efficient.
- **User-Friendly Interface**: Simple and intuitive design for ease of use.
